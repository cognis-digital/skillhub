SkillHub is a local registry for reusable AI agent capabilities, called "skills." Think of it like an app store for your AI agent: you browse a folder of packaged skills, search for the one you need, and install it into your agent with a single command. It automatically handles dependencies — if a skill needs another skill to work, SkillHub installs them both in the right order. It is designed for developers and teams who build or run AI agents and want a reproducible, offline-friendly way to share and reuse agent building blocks without relying on a cloud service.
